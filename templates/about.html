About This Project

The College Timetable Generation System is a web application that uses a Genetic Algorithm to automatically generate conflict-free class timetables for colleges and universities.

How It Works

The system uses an evolutionary approach to solve the timetable scheduling problem:

  1. Initialization — Random timetables are generated as the initial population
  2. Fitness Evaluation — Each timetable is scored based on constraint violations
  3. Selection — The best timetables are selected via tournament selection
  4. Crossover — Parent timetables combine to create offspring
  5. Mutation — Random changes introduce diversity
  6. Repeat — The process continues until a conflict-free timetable is found

Constraints Enforced

No Room Conflicts
Two classes cannot be in the same room at the same time.
No Teacher Conflicts
A teacher cannot teach two classes simultaneously.
No Section Conflicts
A section cannot have two classes at the same time.
Capacity Check
Room capacity must accommodate the number of students.
